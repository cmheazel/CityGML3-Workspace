^|*Requirement  {counter:req-id}* |*/req/Core/AbstractPhysicalSpace*
^|A |The Implementation Specification SHALL contain an element which incorporates the concept defined in the AbstractPhysicalSpace UML class as documented in the AbstractPhysicalSpace section of the <<AbstractPhysicalSpace-section,Core Data Dictionary>>.

^|C |The implementation Specification SHALL provide an element which represents the attributes of the AbstractPhysicalSpace UML class; including the name, definition, type, and cardinality of those documented in the AbstractPhysicalSpace section of the <<AbstractPhysicalSpace-section,Core Data Dictionary>>.
|===
