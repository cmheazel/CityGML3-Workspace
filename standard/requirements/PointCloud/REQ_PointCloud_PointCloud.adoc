[[req_PointCloud_PointCloud]]
[width="90%",cols="2,6"]
|===
^|*Requirement  {counter:req-id}* |*/req/PointCloud/PointCloud* 
^|A |The Implementation Specification SHALL contain an element with the same definition as the PointCloud UML class as documented in the PointCloud section of the <<PointCloud-section,PointCloud Data Dictionary>>.
^|B |The Implementation Specification SHALL represent associations with the same source, target, direction, and cardinalities as those of the PointCloud UML class as documented in the PointCloud section of the <<PointCloud-section,PointCloud Data Dictionary>>.
^|C |The implementation Specification SHALL represent the attributes of the PointCloud UML class; including the name, definition, type, and cardinality of those documented in the PointCloud section of the <<PointCloud-section,PointCloud Data Dictionary>>.
^|D |The implementation Specification SHALL represent the attributes of all parent classes of the the PointCloud UML class; including the name, definition, type, and cardinality of those documented in the PointCloud section of the <<PointCloud-section,PointCloud Data Dictionary>>.
|===
