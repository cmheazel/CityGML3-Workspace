[[req_Tunnel_TunnelInstallation]]
[width="90%",cols="2,6"]
|===
^|*Requirement  {counter:req-id}* |*/req/Tunnel/TunnelInstallation* 
^|A |The Implementation Specification SHALL contain an element with the same definition as the TunnelInstallation UML class as documented in the TunnelInstallation section of the <<TunnelInstallation-section,Tunnel Data Dictionary>>.
^|B |The Implementation Specification SHALL represent associations with the same source, target, direction, and cardinalities as those of the TunnelInstallation UML class as documented in the TunnelInstallation section of the <<TunnelInstallation-section,Tunnel Data Dictionary>>.
^|C |The implementation Specification SHALL represent the attributes of the TunnelInstallation UML class; including the name, definition, type, and cardinality of those documented in the TunnelInstallation section of the <<TunnelInstallation-section,Tunnel Data Dictionary>>.
^|D |The implementation Specification SHALL represent the attributes of all parent classes of the the TunnelInstallation UML class; including the name, definition, type, and cardinality of those documented in the TunnelInstallation section of the <<TunnelInstallation-section,Tunnel Data Dictionary>>.
|===
