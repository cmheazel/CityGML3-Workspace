[[req_Construction_AbstractConstructiveElement]]
[width="90%",cols="2,6"]
|===
^|*Requirement  {counter:req-id}* |*/req/Construction/AbstractConstructiveElement*
^|A |The Implementation Specification SHALL contain an element which incorporates the concept defined in the AbstractConstructiveElement UML class as documented in the AbstractConstructiveElement section of the <<AbstractConstructiveElement-section,Construction Data Dictionary>>.
^|B |The Implementation Specification SHALL provide an element which represents associations with the same source, target, direction, and cardinalities as those of the AbstractConstructiveElement UML class as documented in the AbstractConstructiveElement section of the <<AbstractConstructiveElement-section,Construction Data Dictionary>>.
^|C |The implementation Specification SHALL provide an element which represents the attributes of the AbstractConstructiveElement UML class; including the name, definition, type, and cardinality of those documented in the AbstractConstructiveElement section of the <<AbstractConstructiveElement-section,Construction Data Dictionary>>.
|===
