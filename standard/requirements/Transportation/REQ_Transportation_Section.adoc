[[req_Transportation_Section]]
[width="90%",cols="2,6"]
|===
^|*Requirement  {counter:req-id}* |*/req/Transportation/Section* 
^|A |The Implementation Specification SHALL contain an element with the same definition as the Section UML class as documented in the Section section of the <<Section-section,Transportation Data Dictionary>>.
^|B |The Implementation Specification SHALL represent associations with the same source, target, direction, and cardinalities as those of the Section UML class as documented in the Section section of the <<Section-section,Transportation Data Dictionary>>.
^|C |The implementation Specification SHALL represent the attributes of the Section UML class; including the name, definition, type, and cardinality of those documented in the Section section of the <<Section-section,Transportation Data Dictionary>>.
^|D |The implementation Specification SHALL represent the attributes of all parent classes of the the Section UML class; including the name, definition, type, and cardinality of those documented in the Section section of the <<Section-section,Transportation Data Dictionary>>.
|===
