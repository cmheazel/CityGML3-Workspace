^|*Requirement  {counter:req-id}* |*/req/Transportation/AbstractTransportationSpace*
^|A |The Implementation Specification SHALL contain an element which incorporates the concept defined in the AbstractTransportationSpace UML class as documented in the AbstractTransportationSpace section of the <<AbstractTransportationSpace-section,Transportation Data Dictionary>>.
^|B |The Implementation Specification SHALL provide an element which represents associations with the same source, target, direction, and cardinalities as those of the AbstractTransportationSpace UML class as documented in the AbstractTransportationSpace section of the <<AbstractTransportationSpace-section,Transportation Data Dictionary>>.
^|C |The implementation Specification SHALL provide an element which represents the attributes of the AbstractTransportationSpace UML class; including the name, definition, type, and cardinality of those documented in the AbstractTransportationSpace section of the <<AbstractTransportationSpace-section,Transportation Data Dictionary>>.
