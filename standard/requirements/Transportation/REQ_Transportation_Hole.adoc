[[req_Transportation_Hole]]
[width="90%",cols="2,6"]
|===
^|*Requirement  {counter:req-id}* |*/req/Transportation/Hole* 
^|A |The Implementation Specification SHALL contain an element with the same definition as the Hole UML class as documented in the Hole section of the <<Hole-section,Transportation Data Dictionary>>.
^|B |The Implementation Specification SHALL represent associations with the same source, target, direction, and cardinalities as those of the Hole UML class as documented in the Hole section of the <<Hole-section,Transportation Data Dictionary>>.
^|C |The implementation Specification SHALL represent the attributes of the Hole UML class; including the name, definition, type, and cardinality of those documented in the Hole section of the <<Hole-section,Transportation Data Dictionary>>.
^|D |The implementation Specification SHALL represent the attributes of all parent classes of the the Hole UML class; including the name, definition, type, and cardinality of those documented in the Hole section of the <<Hole-section,Transportation Data Dictionary>>.
|===
